Création de compte

  1. Accéder à l’application

La première fois que vous allez ouvrir Mugi Sells, l’application va vous demander de vous connecter. Vous avez 2 options, indiquer votre adresse email où vous recevrez un code pin ou vous connecter avec un compte google.

Une fois connecté, vous devez indiquer si vous êtes sur l’application pour l’essai ou pour prendre un rendez-vous.

Si vous êtes ici pour l’essai, vous devez lire les conditions générales d’utilisation, si vous les acceptez, vous pouvez cocher la case suivante “En cochant cette case je reconnais avec lu et accepté les CGU.”

Un bouton sera ensuite disponible pour faire une demande d’accès. Cette demande sera automatiquement acceptée et votre essai gratuit de 30 jours commencera.

Paramétrer mon compte

Une fois connecté et en possession d’un accès valide, votre dashboard ressemblera à l’image ci-dessous.

Pour utiliser l’application dans les meilleures conditions possibles, il est très vivement recommandé de renseigner un maximum d'informations sur son entreprise. Ces dernières seront reportées sur vos devis et factures.

Pour cela, cliquez sur le bouton “Commencer”. Un formulaire va s’ouvrir


Vous pouvez renseigner un numéro de facture qui sera utilisé pour vos prochaine facture avec +1, renseignez donc le numéro de votre dernière facture déjà générée si vous souhaitez utiliser cette fonctionnalité. Si ce champ est laissé vide, un numéro sera automatiquement généré pour vos factures.

Vous pouvez renseigner une signature ou un tampon, ce dernier sera reporté sur vos devis.

Si vous souhaitez renseigner ces informations plus tard ou les modifier, vous pourrez les retrouver en cliquant sur l'icône en haut à droite de l’écran qui est soit la photo de profil que vous avez indiqué soit la première lettre de votre nom puis en cliquant sur le bouton “Modifier”.

Page - Dashboard

  1. Votre résumé statistique

La première partie de votre dashboard se compose de 5 statistiques, vous pouvez cliquer sur chacune d’elles pour obtenir la liste des éléments correspondants s' il y en a. Chacun d’eux est divisé en 2 chiffres, le premier est le nombre d’éléments comptés et le deuxième est le montant TTC compté.

  1. Votre prochain rendez-vous

Si vous avez au moins un rendez-vous existant à venir, le plus proche sera affiché ici.

Dans le cas contraire, un message va vous indiquer que vous n’avez aucun rendez-vous à venir.

  1. Vos tâches

Une liste de toutes vos tâches qui sont à réaliser est affichée ici même si leur échéance est passée.

  1. Devis signés

Une liste des devis que vos clients viennent de signer.

Page - Contact

Dans la page contact, vous trouverez une liste de tous ceux que vous avez encodé dans l’application.

Une recherche par nom, numéro de téléphone et adresse est possible.

Vous pouvez encoder un nouveau contact depuis le bouton “Nouveau” en haut à droite qui va ouvrir un formulaire.

Détails - Dans le contact

En cliquant sur un élément de la page contact, vous allez rentrer dans une page qui va vous donner toutes les informations relatives à ce contact que nous allons appeler page de “Détails”.

Cette dernière est divisée en 3 blocs liés à ce contact.

  1. Le premier contient toutes les informations que vous avez renseigné à son sujet, ainsi qu’un bouton “Modifier” qui ouvre un formulaire de modification de ces informations.

  1. Le deuxième contient 4 sous blocs, une navigation est présente pour y naviguer.
  1. Une liste de tous les rendez-vous ainsi qu’un bouton “Nouveau” qui va ouvrir un formulaire de création.
  2. Une liste de tous les projets ainsi qu’un bouton “Nouveau” qui va ouvrir un formulaire de création.
  3. Une liste de toutes les factures ainsi que 2 boutons.
  1. “Facture libre”, vous permet de créer une facture sans passer par un projet. Vous pourrez y ajouter tous les éléments dont vous aurez besoin. (Voir Facture)
  2. “Importer”, vous permet d’importer une facture existante, les fonctionnalités liées seront limitées mais vous pourrez y ajouter des paiements et des avoirs.
  1. Une liste de tous les documents.
  1. Une liste des tâches ainsi qu’un bouton “Nouvelle” qui va ouvrir un formulaire de création.

Détails - Dans le rendez-vous

Dans cette page vous retrouverez toutes les informations relatives au rendez-vous ainsi qu’un bouton “Modifier” qui vous permet de modifier la date mais également de renseigner un statut.

Détails - Dans le projet

Un projet va être l’élément principal autour duquel tout le reste va s’articuler, il contient les éléments que vous souhaitez vendre dans l’onglet “Devis”, la facturation dans l’onglet “Facture” ainsi que l’onglet “Document” qui va regrouper tous les documents liés à ce projet.

Le devis

La première chose que vous allez faire est d'ajouter les éléments que vous allez vendre via un devis.

Pour cela vous pouvez cliquer sur le bouton “Ajouter” qui va vous ouvrir votre catalogue. Si vous avez déjà renseigné des produits, ils seront présents ici, dans le cas contraire vous pouvez cliquer sur “Créer un produit”, un formulaire va s’ouvrir et vous pourrez renseigner le produit dont vous avez besoin.

Choisissez le produit que vous souhaitez ajouter, indiquez la quantité, le taux de tva ainsi que la remise en euro puis en cliquant sur “Soumettre” le produit sera ajouté à votre devis. Vous pouvez ajouter autant de produits que vous le souhaitez.

Un récapitulatif est disponible sur la partie de droite ainsi qu’un bouton qui va vous permettre de générer le devis au format PDF. Si le devis ne s’affiche pas, un bouton, “Télécharger devis” est disponible tout en haut de la page.

Vous pouvez partager le devis avec votre prospect via le bouton “Partager” qui ouvrira votre boite email avec un mail pré-rempli qui contient un lien vers le devis au format PDF mais également le lien de l’application.

Il pourra se connecter à l’application via l’adresse email que vous avez renseignée dans sa fiche client. Il aura accès aux projets, devis, factures, paiements et documents renseignés. Il pourra également signer électroniquement un devis et générer une copie de ce dernier présentant sa signature et la date. Il lui sera également possible de renseigner des documents si nécessaire.

Les devis que vos clients ont signés eux-mêmes sont mis en avant sur votre dashboard.

Vous pouvez également appuyer sur le bouton “Signature”, indiquer la date et si besoin vous pouvez réaliser une signature électronique depuis cet écran si vous êtes avec votre client. Dans le cas d’une signature papier, vous pouvez importer le devis signé dans la page “Documents”.

Les factures

Une fois un devis signé, vous avez accès à la génération de factures.

Il y a 2 chiffres clés

  1. Le restant à facturer qui correspond à

Total TTC du projet - Total des factures TTC

  1. Le restant à encaisser (Solde) qui correspond à

Total TTC du projet - Les encaissements enregistrés

Vous pouvez créer une facture de 3 manières différentes

  1. Facture d’acompte, vous choisissez le pourcentage du projet que vous souhaitez facturer et une facture sera générée
  2. Facture de solde, elle utilise le restant à facturer
  3. Importer une facture, si vous avez généré une facture sur une autre outil, vous pouvez l’importer en indiquant les montants qui lui sont propre

Quand une facture est créée par l’application, elle est au stade de brouillon pour que vous puissiez vérifier les informations et ne possède pas de numéro. Une fois que vous avez vérifié les informations, vous pouvez les valider et une version définitive portant un numéro sera générée.

Attention, si une facture possède un numéro elle ne peut plus être supprimée ou modifiée.

Détails - Dans la facture

Dans une facture vous allez retrouver les informations du client, plusieurs boutons qui permettent de télécharger la version de la facture que vous souhaitez.

Vous allez pouvoir ajouter des paiements et des avoirs, les 2 viendront réduire le solde de votre facture.

Quand une facture est générée elle est au statut de brouillon et ne possède pas de numéro.

Si vous êtes satisfait du document, vous pouvez générer une version définitive qui ne pourra pas être supprimée.

Une fois soldée, vous pouvez générer un duplicata qui contient la liste des paiements et avoir ainsi que la mention “Soldée”.

Pour ajouter un paiement vous pouvez ouvrir un formulaire via le bouton “Ajouter” à côté de “Paiements”

Pour ajouter un avoir vous pouvez ouvrir un formulaire via le bouton “Ajouter” à côté de “Avoir”

En ce qui concerne les factures libres, elles ne sont pas liées à un projet, vous pouvez donc indiquer librement les éléments que vous souhaitez facturer.

En cliquant sur “Ajouter mes éléments”

Vous pouvez choisir le type de facture (Facture de solde est renseigné par défaut).

Vous pouvez renseigner le nom, la quantité, le taux de tva, la remise ainsi que le prix unitaire. Une fois un élément remplit, une coche verte sera présente à côté de son nom. Si un produit n’est pas entièrement renseigné (mise à part la remise) il ne sera pas prit en compte dans les calculs et pas affiché sur la facture.

Une fois vos éléments renseignés, vous pouvez cliquer sur le bouton “valider” en bas à droite, ce qui va créer une version brouillon sans numéro de cette facture.
Si vous êtes satisfait, vous pouvez cliquer sur “Valider ma facture” ce qui va lui attribuer un numéro définitif et elle ne pourra plus être modifiée.

Page - Agenda

Vous trouverez ici la liste de tous vos rendez-vous. Vous pouvez choisir le format d’affichage et cliquer sur un d’eux vous donnera plus d’informations.
Vous trouverez également un bouton permettant de générer des rendez-vous libres.

Page - Devis

Cette page regroupe tous vos devis, vous pouvez soit choisir le statut du devis en cliquant sur l’onglet correspondant, soit cliquer sur “Recherche avancée” et faire une recherche précise

Page - Facturation

Cette page regroupe toutes vos factures et paiements sous la même forme que les devis, vous pouvez choisir d’afficher une liste par type ou via une recherche avancée.

Page - Catalogue

Votre catalogue regroupe tous les produits que vous avez créés.

Vous pouvez en créer de nouveaux depuis le bouton “Nouveau” qui ouvre un formulaire de création.
Vous pouvez modifier n’importe quel produit depuis le bouton “Modifier”. Une modification du nom d’un produit impactera les devis le comprenant, cependant aucune modification de prix n’aura d’impact sur des devis.

A noter que les choix dans les champs “Catégorie” et “Unité” sont ajustables dans la page “Paramètre”

Page - Statistique

Cette page vous donne des statistiques de l’année de votre choix, mois par mois pour les devis signés, les factures, les paiements et les produits vendus. Vous pouvez cliquer sur le mois qui vous intéresse pour obtenir le détail.

A gauche de chaque ligne vous trouverez un symbole.

  1. Flèche verte, le montant TTC mensuel est en hausse par rapport au mois dernier.
  2. Flèche rouge, le montant TTC mensuel est en baisse par rapport au mois dernier.
  3. Sablier, il n’y pas encore de donnée sur le mois.

Pour les produits vendus, vous trouverez en premier une liste de vos produits, en cliquant sur celui de votre choix, vous allez ouvrir un autre tableau qui vous donnera le détail mensuel de ce dernier.

Page - Importation

Cette page vous permet d’exporter des données selon vos choix. Elles seront téléchargées au format CSV et pourront être utilisées sur n’importe quel outil tableur.

Page - Paramètre

Cette page contient un tableau qui regroupe les choix possibles dans les informations au moment de la création ou de la modification de vos produits.

 Il est à noter que vous êtes limité à 30 lignes. Merci de me contacter si vous avez besoin de plus de lignes.

Page - Paramètre rdv

L’application permet à vos prospect de planifier des rendez-vous qui arriveront directement dans votre agenda. Pour se faire vous devez renseigner plusieurs informations et chacune d’elle est indiquée dans un onglet de la page

  • Base
  • Délai jour = C’est un délai entre aujourd’hui et le premier jour qui sera proposé vos prospect comme possibilité de rdv. Ex : Nous sommes le 1er janvier, vous avez indiqué un délai de 10 jours alors le premier jour proposé sera le 11 janvier.
  • Durée des créneaux : La durée des créneaux que vous proposez. Ex : Vous avez indiqué une durée de 15 minutes et votre journée commence à 8h, le premier créneau possible sera à 8h puis à 8h15 puis à 8h30 et ainsi de suite.
  • Type de rdv
  • Vous pouvez créer des types de rendez-vous qui seront sélectionnés par le prospect.
  • La durée sera prise en compte dans la création du rendez-vous
  • La marge : Durée en minute qui indique la marge dont vous avez besoin entre 2 rendez-vous. Ceci sera prit en compte dans les calculs de disponibilités et en cas de conflit, la marge la plus grande sera retenue.
  • Ils peuvent être renseignés dans les rdv que vous allez vous même créer et cela permettra d’indiquer les marges sur chaque rendez-vous et les possibilités de créneaux proposés soient les plus précis possibles (C’est même très fortement recommandé)
  • Période des rdv
  • Indiquez les périodes auxquelles vous souhaitez que vos prospects puissent prendre des rdv (A noter que sur la version d’essai, la période proposée sera limitée à la date de fin de votre essai)
  • Disponibilités des vendeurs
  • Permet d’indiquer les horaires de chaque vendeur de votre équipe, cela jouera sur les horaires proposés aux prospects.
  • Indisponibilités des vendeurs
  • Permet d’indiquer une période où un vendeur ne sera pas disponible

Une fois tous ces éléments renseignés, votre entreprise sera présente dans la liste des entreprises dans la page prise de rendez-vous.
Vous pouvez fournir un lien vers cette application à vos prospects et ils pourront prendre rendez-vous.
